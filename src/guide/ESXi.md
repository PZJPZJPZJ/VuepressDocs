# ESXi:Vmware虚拟机系统
## 安装教程
1. 使用工具写入ISO镜像到磁盘中
2. 进入磁盘引导等待倒计时结束
3. 按照提示完成安装流程
## 修改默认虚拟闪存占用
1. 进入安装引导后倒计时内按下Shift+O
2. 在原有命令后加空格输入以下内容(可调整分配空间推荐8192M)
```
autoPartitionOSDataSize=8192
```
3. 输入完成按下回车进入正常安装流程
